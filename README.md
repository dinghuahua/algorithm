tcp 和 udp 的区别

tcp是点到点，是面向连接的，它侧重连接的安全性和可靠性，所以在正式收发报文之前，会先确认两端的发送和接受能力，并保证传输的完整
udp是面向报文的，服务器可以同时向多个客户端发送数据，并且不需要建立连接，也不验证传输的完整性，所以它不安全，但是传输速度快 ，只受网络带宽，速率等影响
